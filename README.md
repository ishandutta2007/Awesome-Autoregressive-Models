# Awesome-Autoregressive-Models
## Autoregressive Generative Models

Autoregressive (AR) models are a class of generative AI that predict the next element in a sequence—like a word or a pixel—based on all preceding elements. Because they model joint probability explicitly using the chain rule, they are the powerhouses behind modern text, code, and music generation.

## Top Autoregressive Generative Models

Modern AR models generally use the Transformer architecture to map long-range dependencies efficiently. The most prominent examples include:

| Model | Year | Paper / Source | Description |
| :--- | :--- | :--- | :--- |
| **GPT-4** | 2023 | [GPT-4 Technical Report](https://arxiv.org/abs/2303.08774) | Developed by [OpenAI](https://openai.com), this is the flagship autoregressive model for natural language understanding, complex reasoning, and code generation. |
| **Gemini** | 2023 | [Gemini: A Family of Highly Capable Multimodal Models](https://arxiv.org/abs/2312.11805) | Google’s flagship multimodal model family, built natively to integrate text, audio, images, and video in an autoregressive generation loop. |
| **Claude 3.5** | 2024 | [Claude 3.5 Sonnet Model Card](https://www.anthropic.com/news/claude-3-5-sonnet) | Developed by [Anthropic](https://anthropic.com), this state-of-the-art autoregressive language model is highly regarded for its coding proficiency and nuanced text generation. |
| **Llama 3** | 2024 | [The Llama 3 Herd of Models](https://arxiv.org/abs/2407.21783) | Released by [Meta AI](https://meta.com), this is one of the most powerful openly available autoregressive language models, widely used for fine-tuning across enterprise and consumer applications. |

## Legacy & Foundational AR Models

| Model | Year | Paper / Source | Description |
| :--- | :--- | :--- | :--- |
| **BERT** | 2018 | [BERT: Pre-training of Deep Bidirectional Transformers for Language Understanding](https://arxiv.org/abs/1810.04805) | A foundational autoregressive/masked language model developed by [Google Research](https://blog.google) that set the stage for bidirectional contextual analysis in NLP. |
| **XLNet** | 2019 | [XLNet: Generalized Autoregressive Pretraining for Language Understanding](https://arxiv.org/abs/1906.08237) | An autoregressive model designed to overcome some of the limitations of older models by maximizing expected likelihood over all permutations of the factorization order. |
