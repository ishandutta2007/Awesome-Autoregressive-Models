# Awesome-Autoregressive-Models
## Autoregressive Generative Models

Autoregressive (AR) models are a class of generative AI that predict the next element in a sequence—like a word or a pixel—based on all preceding elements. Because they model joint probability explicitly using the chain rule, they are the powerhouses behind modern text, code, and music generation.

## Top Autoregressive Generative Models

Modern AR models generally use the Transformer architecture to map long-range dependencies efficiently. The most prominent examples include:

* **GPT-4:** Developed by [OpenAI](https://openai.com), this is the flagship autoregressive model for natural language understanding, complex reasoning, and code generation. 
* **Gemini:** Google’s flagship multimodal model family, built natively to integrate text, audio, images, and video in an autoregressive generation loop. 
* **Claude 3.5:** Developed by [Anthropic](https://anthropic.com), this state-of-the-art autoregressive language model is highly regarded for its coding proficiency and nuanced text generation.
* **Llama 3:** Released by [Meta AI](https://meta.com), this is one of the most powerful openly available autoregressive language models, widely used for fine-tuning across enterprise and consumer applications.

## Legacy & Foundational AR Models

* **BERT:** A foundational autoregressive/masked language model developed by [Google Research](https://blog.google) that set the stage for bidirectional contextual analysis in NLP.
* **XLNet:** An autoregressive model designed to overcome some of the limitations of older models by maximizing expected likelihood over all permutations of the factorization order.
