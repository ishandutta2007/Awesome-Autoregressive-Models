## 📖 Introduction to Autoregressive Generative Models

Autoregressive (AR) models are a class of generative AI that predict the next element in a sequence—like a word or a pixel—based on all preceding elements. By modeling joint probability explicitly via the chain rule, they serve as the foundational architecture for modern text, code, and multimodal generation.
